Sobre From Zero to Java

Este site foi criado e é mantido por Aurélio Akira M. Matsui e alguns amigos. Ele dá bastante importância à sua privacidade mas quer ter certeza que ninguém usará o material neste site para cobrar dinheiro de terceiros. Esta é a única razão pela qual ele decidiu escrever o seu nome nesta página.

Liberdade

Todo o conteúdo é livre e vai permanecer assim para sempre. Visite licença para mais informações.

FromZeroToJava é um curso de Java para aqueles que não sabem nada de programação. Nosso método é baseado em processos e ferramentas com comprovação prática. Junto com a linguagem Java, nós também apresentamos as melhores ferramentas de Software Livre com as quais todos os programadores Java deveriam estar familiarizados.

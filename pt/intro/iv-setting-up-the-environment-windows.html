Passo 1 - Instalando VirtualBox e Vagrant no Windows

Introdução

Instalar o ambiente em uma máquina Windows gasta mais tempo e é mais tedioso do que no Linux, pois você precisa baixar e instalar manualmente algumas partes do software. As telas abaixo foram uma contribuição de Priscila.

Instalar o plugin snapshot no Vagrant

No DOS ou no PowerShell, digite o seguinte:

vagrant plugin install vagrant-vbox-snapshot

Fazer o download do Putty

Na verdade, não há instalação. Você precisa simplesmente copiar o endereço no seu computador e rodá-lo no seu PC.

http://www.chiark.greenend.org.uk/~sgtatham/putty/download.html

Configure o Putty para logar automaticamente na sua máquina virtual.

https://github.com/Varying-Vagrant-Vagrants/VVV/wiki/Connect-to-Your-Vagrant-Virtual-Machine-with-PuTTY
